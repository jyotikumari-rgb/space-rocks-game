
# 🚀 Space Rocks Game

A 2D space shooter game built with Python and Pygame.

## 🎮 Features

- Spaceship controls: rotate, accelerate, shoot
- Asteroid collisions and splits
- Win/lose messages
- Simple sprites and smooth gameplay

## 🕹️ Controls

- Arrow keys: Move/rotate spaceship
- Space: Shoot bullets
- Q / ESC: Quit

## 🛠️ Requirements

- Python 3.10+
- Pygame 2.6+
